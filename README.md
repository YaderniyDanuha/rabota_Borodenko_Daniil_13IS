Задача 2. Калькулятор скидки в магазине (if-elif-else + арифметика)
Условие:
В магазине действует система скидок:
- Если сумма меньше 500 рублей — скидка 0%
- От 500 до 2000 рублей — скидка 5%
- От 2001 до 5000 рублей — скидка 10%
- Больше 5000 рублей — скидка 15%
С клавиатуры вводится сумма покупки.
Выведите две строки:
Скидка: X рублей
К оплате: Y рублей
Подсказка по проценту:
Чтобы найти 1% от числа — раздели число на 100 (summa // 100).
Чтобы найти 5% — умножь 1% на 5.
Ввод	Вывод
2500	Скидка: 250 рублейК оплате: 2250 рублей
400	Скидка: 0 рублейК оплате: 400 рублей
6000	Скидка: 900 рублейК оплате: 5100 рублей

Задача 6. Прямоугольный треугольник
Условие:
Введите три положительных числа — длины сторон треугольника.
Определите, может ли существовать прямоугольный треугольник.
Справочный материал (теорема Пифагора):
В прямоугольном треугольнике квадрат гипотенузы (самой длинной стороны) равен сумме квадратов двух других сторон:
a² + b² = c² (где c — гипотенуза, a, b - катеты).
Ввод	Вывод
345	Да
556	Нет
81517	Да
